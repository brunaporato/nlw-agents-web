## NLW Agents - web
This is the frontend of NLW Agents, a platform where students can interact with an AI assistant that answers their questions in real time based on the content of ongoing live classes.

## Tech Stack
- React – Modern frontend library
- Vite – Fast development build tool
- TailwindCSS – Utility-first CSS framework
- ShadCN UI – Accessible and themeable UI components
- React Router DOM – Client-side routing
- React Query – Data fetching and caching
- TypeScript – Type-safe development experience
